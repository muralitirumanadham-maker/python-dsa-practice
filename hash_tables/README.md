# Hash Tables

This folder contains Python implementations of Hash Table (Hash Map) concepts
and interview-focused problems.

Hash tables store data in key-value pairs and provide efficient average-case
time complexity for insertion, deletion, and lookup.

---

## Topics Planned

- Hash table basics
- Hash functions
- Collision handling
- Chaining
- Open addressing
- Hash table implementation
- Common interview problems using hash maps

---

## Interview Value

Hash tables are one of the most important data structures and are heavily used
in technical interviews at companies like Google, Amazon, and Microsoft.
