# Hash Tables

This folder contains Python implementations of Hash Table (Hash Map) concepts
and interview-focused problems.

Hash tables store data in key-value pairs and provide efficient average-case
time complexity for insertion, deletion, and lookup.

---

## Implementations

1. Hash Table using Chaining
   - Custom hash function
   - Collision handling with chaining
   - Insert (set)
   - Search (get)
   - Retrieve all keys

3. Find Duplicate Elements

4. First Non-Repeating Character
   - Frequency-based hash map solution
   - Returns first unique character in a string

5. Group Anagrams
   - Group strings with same character frequency
   - Hash map with sorted string as key
