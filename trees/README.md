# Trees

This folder contains Python implementations of Tree data structures and
interview-focused problems.

Trees are hierarchical data structures widely used in searching, sorting,
and representing structured data.

---

## Implementations

1. Binary Search Tree (BST)
   - Insert nodes
   - Search (contains)
   - Print tree structure

---

## Topics Planned

- Binary Tree basics
- Tree traversals (Inorder, Preorder, Postorder)
- Level Order Traversal (BFS)
- Binary Search Tree advanced operations
- Common interview problems on trees

---

## Interview Value

Tree-based problems are heavily asked in technical interviews at companies
like Google, Amazon, and Microsoft.
