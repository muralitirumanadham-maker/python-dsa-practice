# Linked List

This folder contains Python implementations of important Linked List problems for interview preparation.

## Implementations

1. Linked List Basics (Node, append, pop, reverse)
2. Reverse Linked List (Iterative)
3. Middle of Linked List
4. Merge Two Sorted Lists
5. Remove Nth Node from End
6. Linked List Cycle Detection
7. Palindrome Linked List
8. Add Two Numbers
9. Reorder List
10. Reverse Nodes in K Groups

## Concepts Covered
- Pointer manipulation
- Slow and fast pointer technique
- Two pointer technique
- In-place reversal
- Linked list traversal
